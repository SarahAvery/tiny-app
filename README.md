## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- @mapbox/node-pre-gyp,
- node-gyp
- body-parser
- cookie-session
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm start` command.
- Set by default to run on localhost:8080
